# Affordable Wedding - TikTok LIVE Wedding Platform

A modern, responsive wedding booking platform for TikTok LIVE wedding broadcasts by ASKOFUU AUKA.

## 🌟 Features

### For Couples
- **Live Wedding Booking**: Book your TikTok LIVE wedding ceremony
- **Photo Upload**: Share up to 5 beautiful photos for your ceremony backdrop
- **Wedding Vibes**: Choose from High Energy Praise, Romantic & Soft, or Traditional styles
- **Legal Transparency**: Clear legal notices and terms of engagement

### For Brands
- **Partnership Opportunities**: Reach 50K+ live viewers and generate 1M+ likes per event
- **Follower Growth**: Massive brand exposure and follower acquisition
- **Global Reach**: Expand to 50+ countries with authentic engagement

### Admin Dashboard
- **Secure Authentication**: Encrypted password system with session management
- **Booking Management**: View and respond to wedding bookings
- **Brand Inquiries**: Manage partnership requests
- **Email Integration**: Send responses directly from the dashboard

## 🚀 Technology Stack

- **Framework**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS with custom animations
- **Authentication**: bcrypt with secure session management
- **Form Handling**: React Hook Form with Zod validation
- **Icons**: Lucide React
- **Animations**: Framer Motion for smooth transitions
- **Email**: EmailJS integration ready

## 🎨 Design Features

- **Responsive Design**: Mobile-first approach with beautiful animations
- **Color Palette**: Royal Purple, Deep Rose, and Shimmer Gold
- **Background Slideshow**: Dynamic wedding photo carousel
- **Interactive Elements**: Smooth transitions and hover effects
- **Professional UI**: Clean, modern interface with accessibility features

## 🔐 Security

- **Password Encryption**: bcrypt hashing with 12 salt rounds
- **Session Management**: 24-hour session expiration with activity extension
- **Route Protection**: Secure admin dashboard with authentication guards
- **Input Validation**: Comprehensive form validation and sanitization

## 📱 Admin Access

- **URL**: `/auka`
- **Default Credentials**: 
  - Username: `admin`
  - Password: `ASKOFU_ADMIN_2026`
- **Features**: Change password, manage bookings, respond to inquiries
